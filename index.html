Cloud Sync — Firebase Firestore

Data tersinkron otomatis ke semua device via Firebase (gratis)

Belum terhubung

📋 Cara Setup Firebase (1x saja, gratis):

  1. 1.Buat project di Firebase Console lalu aktifkan Firestore Database
  2. 2.Di tab Rules Firestore, set allow read, write: if true (akses publik, tanpa key)
  3. 3.Isi FIREBASE_PROJECT_ID di js/config.js dengan ID project Firebase Anda
  4. 4.Reload halaman ini — koneksi terhubung otomatis, tidak perlu isi apa pun di sini
⚡ Smart-Sync: ON
⚡

Smart-Sync Aktif — Sinkronisasi Otomatis & Cerdas

  • Data dikirim ke cloud hanya jika ada perubahan (penambahan, penghapusan, perubahan password, user baru/dihapus)
  • Jika tidak ada perubahan, tidak ada request ke Firestore — hemat kuota & tidak membuang bandwidth
  • Pull otomatis (Smart-Sync) berjalan setiap kali app dibuka; antar device tidak ada live-update instan seperti Realtime — cukup buka ulang tab / klik "Tarik Data dari Cloud"
  • Hash-check mencegah push duplikat & pull yang tidak perlu
Terakhir sync: -

Klik "Muat Log" untuk melihat riwayat perubahan

📄 Security Rules Firestore yang dipakai (klik untuk lihat)
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
